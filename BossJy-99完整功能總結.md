## 🔧 GLM-4.7 API 設置

### 設置 API Key

#### 方法 1：localStorage
```javascript
// 在瀏覽器控制台執行
localStorage.setItem('GLM_API_KEY', 'your-api-key-here')
```

#### 方法 2：環境變量
```bash
# .env.local
GLM_API_KEY=your-api-key-here
```

#### 方法 3：應用內設置
```typescript
import { setApiKey } from '@/lib/boss-jy-99-api'

// 設置 API Key
setApiKey('your-api-key-here')
```
