## 🚀 如何生成剩余產品圖片

### 方法一：批量生成所有圖片
```bash
bun run scripts/generate-extended-images.ts
```
這個腳本會自動生成所有 35 個新產品的圖片。

### 方法二：逐個生成特定產品圖片
```bash
# 生成特定產品圖片
bun run scripts/generate-single-image.ts [filename]

# 可用文件名列表
bun run scripts/generate-single-image.ts [按 tab 鍵查看]
```
