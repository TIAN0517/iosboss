# LinLauncher.exe 問題排查指南

## 🔍 問題描述

- **問題**：`LinLauncher.exe` 容量變小（795KB）且無法啟動
- **正常情況**：可執行文件通常應該有正常大小（通常 > 1MB）

---

## 🔧 可能原因

### 1. 文件損壞或被截斷
- 文件在傳輸或複製過程中損壞
- 磁盤錯誤導致文件不完整
- 文件被意外截斷

### 2. 防毒軟件誤刪或隔離
- Windows Defender 或其他防毒軟件誤判為威脅
- 文件被隔離或部分刪除
- 需要檢查防毒軟件日誌

### 3. 構建過程不完整
- 構建過程被中斷
- 編譯錯誤導致生成不完整的文件
- 需要重新構建

### 4. 文件被覆蓋為不完整版本
- 下載或複製過程中網絡中斷
- 文件被不完整版本覆蓋

---

## ✅ 解決方案

### 方案 1: 檢查文件完整性

```powershell
# 檢查文件是否存在
Test-Path "LinLauncher.exe"

# 檢查文件大小
(Get-Item "LinLauncher.exe").Length

# 檢查文件哈希（如果知道原始哈希）
Get-FileHash "LinLauncher.exe" -Algorithm SHA256
```

### 方案 2: 檢查防毒軟件

1. **Windows Defender**：
   - 打開「Windows 安全性」
   - 檢查「病毒與威脅防護」→「保護記錄」
   - 查看是否有 `LinLauncher.exe` 被隔離

2. **其他防毒軟件**：
   - 檢查防毒軟件的隔離區
   - 查看是否有相關警告或日誌

### 方案 3: 重新獲取文件

1. **如果有備份**：
   - 從備份恢復原始文件
   - 檢查版本控制系統（Git）是否有原始文件

2. **如果從源代碼構建**：
   - 清理構建目錄
   - 重新構建項目
   - 確保構建過程完整

3. **如果從下載獲取**：
   - 重新下載文件
   - 確保下載完整
   - 檢查下載文件的哈希值

### 方案 4: 檢查文件位置

```powershell
# 搜索項目中的所有 .exe 文件
Get-ChildItem -Path . -Filter "*.exe" -Recurse | Select-Object FullName, Length, LastWriteTime

# 檢查常見位置
$paths = @(
    ".\LinLauncher.exe",
    ".\dist\LinLauncher.exe",
    ".\build\LinLauncher.exe",
    ".\out\LinLauncher.exe",
    ".\release\LinLauncher.exe",
    ".\bin\LinLauncher.exe"
)
foreach ($path in $paths) {
    if (Test-Path $path) {
        Write-Host "找到: $path"
    }
}
```
