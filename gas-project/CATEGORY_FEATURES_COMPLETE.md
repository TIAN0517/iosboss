## 🎯 API 數據流程

### 分類 API
```
GET /api/categories
Response: Category[]
{
  id, name, slug, icon, sortOrder
}
```

### 產品 API
```
GET /api/products
Response: Product[]
{
  id, name, description, price, imageUrl,
  stock, categoryId, featured, sortOrder,
  views, sales, rating
}

GET /api/products?categoryId={id}
Response: Product[] (指定分類的產品)

GET /api/products?featured=true
Response: Product[] (特色產品)
```
