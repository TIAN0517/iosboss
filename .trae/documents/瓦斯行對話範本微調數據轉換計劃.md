# 瓦斯行對話範本轉 Ollama 微調數據計劃

## 📋 目標
將台灣瓦斯行常見問題對話範本轉換為 Ollama 模型微調用的訓練數據格式,強化模型的瓦斯業專業知識。

## 🔧 執行步驟

### 1. 創建訓練數據目錄結構
- 創建 `line_bot_ai/training_data/` 目錄
- 建立分類子目錄: `dialogues/`, `maintenance/`, `safety/`

### 2. 轉換對話範本為 JSONL 格式
為每個對話範本創建標準化的訓練樣本:
```jsonl
{
  "system": "瓦斯產業專用 AI 總控核心",
  "user": "瓦斯鋼瓶押金多少錢?",
  "assistant": "恁好！一般10公斤鋼瓶押金500元，20公斤800元。退押金請帶鋼瓶、押金收據、身份證，到原購買瓦斯行辦理哦！",
  "metadata": {
    "category": "basic_consultation",
    "taiwanese_annotated": true,
    "voice_optimized": true
  }
}
```

### 3. 創建維修知識專門訓練集
- 爐具故障與維修 (15+ 樣本)
- 熱水器故障與維修 (15+ 樣本)
- 配管安裝與維修 (10+ 樣本)

### 4. 整合台語語音優化
- 在 `prompts.py` 中新增台語語音提示詞模組
- 添加 TTS 播放優化指令
- 標註台語關鍵詞發音規則

### 5. 生成微調腳本
- 創建 `ollama_fine_tuning.py` 腳本
- 支援批量生成訓練數據
- 提供數據驗證和格式檢查

### 6. 創建模型微調指南
- 建立 `OLLAMA_FINE_TUNING_GUIDE.md`
- 包含微調命令、參數配置
- 提供測試和驗證步驟

## 📦 預期成果
- **對話範本數據集**: 50+ 高品質對話樣本
- **維修知識數據集**: 40+ 技術維修樣本
- **台語語音優化**: 完整的台語關鍵詞和 TTS 配置
- **微調腳本**: 自動化訓練數據生成和驗證工具

## 🎯 技術特點
- 兼容現有 `prompts.py` 架構
- 支援 ASR 語音輸入強化
- 保留角色切換功能 (客服/主管/技術)
- 整合台語專業詞彙標註
