# 員工註冊系統使用指南

## 📍 如何訪問員工註冊系統

### 方法 1：直接訪問 URL（最簡單）

在瀏覽器中直接輸入：
```
http://your-domain/register
```

例如：
- 本地開發：`http://localhost:9999/register`
- 生產環境：`https://your-domain.com/register`

### 方法 2：從登入頁面進入

1. 訪問登入頁面：`/login`
2. 在登入頁面底部找到「員工帳號申請？」連結
3. 點擊「點擊這裡申請」即可進入註冊頁面

### 方法 3：管理員在員工管理頁面創建（需要登入）

1. 以管理員身份登入系統
2. 進入「員工」管理頁面
3. 點擊「新增員工」按鈕
4. 填寫員工資訊並創建帳號

## 🎯 註冊功能說明

### 員工自行申請（公開，無需登入）

- **訪問地址**：`/register`
- **適用對象**：新員工
- **審核流程**：
  1. 員工填寫註冊表單
  2. 提交申請
  3. 帳號創建但預設為「未啟用」狀態
  4. 管理員在員工管理頁面啟用帳號
  5. 員工可以登入使用

### 管理員創建帳號（需要管理員權限）

- **訪問地址**：登入後 → 「員工」頁面 → 「新增員工」
- **適用對象**：管理員
- **特點**：創建後帳號立即啟用，無需審核

## 📝 註冊表單欄位

| 欄位 | 必填 | 說明 |
|------|------|------|
| 姓名 | ✅ | 真實姓名 |
| 帳號 | ✅ | 登入使用的帳號（唯一） |
| 電子信箱 | ✅ | 用於接收通知（唯一） |
| 手機號碼 | ❌ | 聯絡電話 |
| 部門 | ❌ | 所屬部門 |
| 職務 | ✅ | 一般員工 或 司機 |
| 密碼 | ✅ | 至少 6 個字符 |
| 確認密碼 | ✅ | 需與密碼一致 |

## ✅ 註冊成功後

1. 系統會顯示「申請成功」訊息
2. 自動跳轉到登入頁面
3. 提示「請等待管理員審核啟用您的帳號」
4. 管理員在員工管理頁面啟用帳號後，即可登入

## 🔧 管理員啟用帳號步驟

1. 以管理員身份登入系統
2. 進入「員工」管理頁面
3. 找到未啟用的員工（`isActive: false`）
4. 點擊「啟用」按鈕
5. 員工即可使用帳號登入

## ❓ 常見問題

### Q: 為什麼註冊後無法登入？

A: 註冊後帳號預設為「未啟用」狀態，需要管理員審核啟用。請聯繫管理員啟用您的帳號。

### Q: 忘記密碼怎麼辦？

A: 目前系統暫不支持密碼重置功能，請聯繫管理員重置密碼。

### Q: 可以修改註冊資訊嗎？

A: 註冊後如需修改資訊，請聯繫管理員在員工管理頁面進行修改。

### Q: 註冊頁面無法訪問？

A: 請確認：
1. URL 是否正確：`/register`
2. 服務是否正常運行
3. 是否已重新構建 Docker 鏡像（如果使用 Docker）

## 🔗 相關 API

- **員工自行註冊**：`POST /api/auth/self-register`
- **管理員創建員工**：`POST /api/auth/register`（需要管理員權限）
- **員工列表**：`GET /api/staff`
- **啟用/停用員工**：`PUT /api/staff/[id]`
