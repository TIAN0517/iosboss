## 🔧 系統配置

### GLM-4.7 API Key

1. 設置環境變量：
```bash
export GLM_API_KEY="your-api-key"
```

2. 或在 localStorage 中設置：
```javascript
localStorage.setItem('GLM_API_KEY', 'your-api-key')
```
